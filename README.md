# AIVerse 🤖

> Your intelligent, self-updating guide to the AI ecosystem.

AIVerse is an open-source FastAPI application that helps developers, researchers, and enthusiasts:
- Discover and compare AI tools, LLMs, and agents
- Understand strengths, weaknesses, and ideal use cases for each
- Stay updated automatically with the latest AI releases and news  

## Features
- 🔍 Search AI tools by use case, category, or capability
- ⚖️ Side-by-side comparison of any two AI tools
- 🧠 AI-powered explanations (via LLM integration)
- 📰 Auto-updating feed of new AI tools and releases
- 🏷️ Categorized: LLMs, Agents, Image Gen, Code Gen, Voice, Multimodal, etc.
- ✅ Open-source and community-driven
